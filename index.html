This application was designed and developed by Esri's applications prototype lab in Redlands, California. The applicatio is build using:

  • ArcGIS API for JavaScript (beta) by Esri
  • jQuery
  • D3 by
  • Sylvester by James Coglan
  • Google Fonts Roboto and Berkshire Swash

The Earthquake data consists of 10835 seimic events from 1900 with magnitutudes greater than six on the Richter scale.

Seimic events hightlighted red in the chart indicate those events within a one year period, or half a year either side of the red line.
